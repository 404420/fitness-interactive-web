# Exercise animations

Place licensed exercise motion files here and reference them from `exerciseMedia` in `index.html`.

Expected MVP filenames:

- `bench-press.webm`
- `back-squat.webm`
- `push-up.webm`
- `lat-pulldown.webm`
- `kettlebell-swing.webm`

Use WebM or MP4 when possible. GIF also works, but is heavier. For every file, keep `sourceName`, `sourceUrl`, `license`, and `attributionText` in the exercise metadata. Do not add random Google images or videos unless the license explicitly allows this use.
